# Исправление проблем с бэкендом

## Проблемы и решения

### 1. Проблема с PowerShell и кодировкой
**Проблема:** PowerShell не может выполнить команды из-за кодировки в путях.

**Решение:** Используйте скрипт `check_backend.py`:
```bash
python check_backend.py
```

### 2. Отсутствие тестовых данных
**Проблема:** API возвращает пустые результаты.

**Решение:** Скрипт автоматически создаст:
- Суперпользователя (admin/admin123)
- Тестовые услуги
- Информацию о фотографе
- Контактные данные

### 3. Проверка работы API
**Проблема:** Неизвестно, работают ли endpoints.

**Решение:** Запустите тест:
```bash
python test_api.py
```

## Пошаговая инструкция

### Шаг 1: Исправление бэкенда

python check_backend.py

## Возможные проблемы

### CORS ошибки
Если фронтенд не может подключиться к API:
1. Убедитесь, что бэкенд запущен на порту 8000
2. Проверьте настройки CORS в `drf/settings.py`

### Пустые данные
Если API возвращает пустые результаты:
1. Запустите `check_backend.py` для создания тестовых данных
2. Проверьте админ панель: http://localhost:8000/admin/

### Ошибки JavaScript
Если фронтенд показывает ошибки:
1. Откройте консоль браузера (F12)
2. Проверьте, что API доступен
3. Убедитесь, что все компоненты импортированы

## Контакты для поддержки

Если проблемы остаются:
1. Проверьте логи в консоли браузера
2. Проверьте логи Django сервера
3. Убедитесь, что все зависимости установлены
