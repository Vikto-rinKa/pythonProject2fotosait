# Руководство по навигации

## Что было улучшено

### 1. Плавные переходы между страницами
- Добавлен компонент `PageTransition` для плавных анимаций при смене страниц
- Каждая страница теперь появляется с анимацией fade-in
- Автоматическая прокрутка к началу страницы при навигации

### 2. Улучшенное навигационное меню
- Анимированные кнопки с эффектом "блеска" при наведении
- Поднятие кнопок при hover эффекте
- Улучшенные активные состояния
- Плавные переходы для всех интерактивных элементов

### 3. Индикатор загрузки
- Красивый спиннер при переходах между страницами
- Показывается на 300мс для имитации загрузки
- Улучшает восприятие скорости приложения

### 4. Хлебные крошки
- Навигационные хлебные крошки на всех страницах (кроме главной)
- Показывают текущее местоположение пользователя
- Позволяют быстро вернуться на предыдущие страницы

### 5. Улучшенные стили
- Удалены дублирующиеся CSS правила
- Добавлены современные анимации
- Улучшена типографика и spacing
- Адаптивный дизайн для мобильных устройств

## Как использовать

### Навигация
- Кликните на любую кнопку в меню для перехода на соответствующую страницу
- При переходе автоматически прокручивается к началу страницы
- Используйте хлебные крошки для быстрого возврата на предыдущие страницы

### Анимации
- Все переходы теперь плавные и анимированные
- Кнопки реагируют на наведение с красивыми эффектами
- Контент страниц появляется с анимацией fade-in

### Мобильная версия
- Навигация адаптирована для мобильных устройств
- Кнопки меню переносятся на новую строку при необходимости
- Сохраняется вся функциональность на всех размерах экрана

## Технические детали

### Компоненты
- `PageTransition` - обертка для анимированных переходов
- `LoadingIndicator` - индикатор загрузки
- `Breadcrumbs` - навигационные хлебные крошки

### CSS классы
- `.page-transition` - анимации переходов
- `.menu a` - стили навигационных кнопок
- `.breadcrumbs` - стили хлебных крошек
- `.loading-indicator` - стили индикатора загрузки

### Анимации
- `fadeInUp` - анимация появления контента
- `spin` - анимация спиннера загрузки
- Hover эффекты для всех интерактивных элементов
